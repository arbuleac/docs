---
title: "Add router"
description: "Instructions to add a router device. "
---

Login to DNS Filtering platform into your account. Click on "Add a device" button.

![Devices Add Router 1 Pn](/images/devices_add_router_1.png)

Enter device name, select type to "Router" and click "Next"

![Devices Add Router 2 Pn](/images/devices_add_router_2.png)

Device is added and you can configure device now ("Configure now") or you can configure it later ("I will do this later")

![Devices Add Router 3 Pn](/images/devices_add_router_3.png)

The new router device in now available in devices list and by default protection is enabled and you can change protection configuration at any time ("Configure")

![Devices Add Router 4 Pn](/images/devices_add_router_4.png)
