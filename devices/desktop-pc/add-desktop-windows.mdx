---
title: "Add desktop/notebook (Windows)"
description: "Instructions to add a new desktop or notebook device running Windows operating systems."
---

_Prerequisites: HeyShield account, logged in._

Go to HeyShield page, section "Protect" and click on "Add a device".

![Devices Add Desktop 1 Pn](/images/devices_add_desktop_1.png)

Fill in "Device name" (e.g Desktop PC), select from Type drop-down list "Windows 10/11" and click "Next"

![Devices Add Desktop 3 Pn](/images/devices_add_desktop_3.png)

Download HeyShield application for Windows from displayed URL

![Devices Add Desktop 5 Pn](/images/devices_add_desktop_5.png)

After download is complete please click to install HeyShield application

![Devices Add Desktop 7 Pn](/images/devices_add_desktop_7.png)

Install HeyShield application

![Devices Add Desktop 8 Pn](/images/devices_add_desktop_8.png)

Confirm or change default path where HeyShield application will be installed and click "Next"

![Devices Add Desktop 9 Pn](/images/devices_add_desktop_9.png)

"Next"

![Devices Add Desktop 10 Pn](/images/devices_add_desktop_10.png)

"Next"

![Devices Add Desktop 11 Pn](/images/devices_add_desktop_11.png)

wait for application to be installed completly

![Devices Add Desktop 12 Pn](/images/devices_add_desktop_12.png)

Installation completed, click "Finish"

![Devices Add Desktop 13 Pn](/images/devices_add_desktop_13.png)

click "Continue" on HeyShield platform, and copy the activation code

![Devices Add Desktop 14 Pn](/images/devices_add_desktop_14.png)

Start HeyShield application and allow application to make changes

![Devices Add Desktop 15 Pn](/images/devices_add_desktop_15.png)

click on "Start setup"

![Devices Add Desktop 16 Pn](/images/devices_add_desktop_16.png)

enter activation (PIN) code and click "Verify"

![Devices Add Desktop 17 Pn](/images/devices_add_desktop_17.png)

If code is validated, application is complete configured and protection automaticaly enabled

![Devices Add Desktop 18a Pn](/images/devices_add_desktop_18a.png)

You can disable filtering protection from HeyShield application

![Devices Add Desktop 18b Pn](/images/devices_add_desktop_18b.png)

Return to HeyShield platform interface, where after code validation device is added to devices list, and you can configure it now ...

![Devices Add Desktop 19 Pn](/images/devices_add_desktop_19.png)

... or later by selecting device from devices list. 

![Devices Add Desktop 20 Pn](/images/devices_add_desktop_20.png)
