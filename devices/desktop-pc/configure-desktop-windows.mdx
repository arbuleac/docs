---
title: "Configure desktop/notebook (Windows)"
description: "Instructions to configure desktop or notebook device running Windows operating systems."
---

From HeyShield platform, section "Protect" you can manage all devices added. To disabled/enable DNS filtering, just swipe button for desired device. To configure a device from list, just click on "Configure" link under desired device.

![Devices Configure Desktop 1 Pn](/images/devices_configure_desktop_1.png)

Configuration page is splitted in few sections:

- Filtering - where service can be enabled/disabled for managed device
- Categories - where for most common predefined categories you can enable/disable filtering protection

![Devices Configure Desktop 2 Pn](/images/devices_configure_desktop_2.png)

more predefined categories

![Devices Configure Desktop 3 Pn](/images/devices_configure_desktop_3.png)

- Application - where for most common predefined applications you can enable/disable filtering protection

![Devices Configure Desktop 4 Pn](/images/devices_configure_desktop_4.png)

- Custome rules - where you can create custom rules for any URL (domain or subdomain)

![Devices Configure Desktop 6 Pn](/images/devices_configure_desktop_6.png)

To create a custom rule enter domain and select allow or block traffic 

![Devices Configure Desktop 7 Pn](/images/devices_configure_desktop_7.png)

Custom rules will be listed in "Custom domain filter" section

![Devices Configure Desktop 8 Pn](/images/devices_configure_desktop_8.png)

- Delete device - if device does not exist anymore you can delete it from list of devices by click on "Delete", and then confirm deletion.

![Devices Configure Desktop 9 Pn](/images/devices_configure_desktop_9.png)
